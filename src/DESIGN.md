# Design System Theme

This document outlines the core aesthetic and functional properties of our design system.

## Color Palette

Our system operates in a **dark** color mode, prioritizing visual comfort and energy efficiency for users.

### Primary Color
The primary brand color is `#f4d125`. This vibrant yellow is used for key interactive elements like buttons, calls-to-action, and primary headings to draw immediate attention and establish brand identity.

### Secondary Color
The secondary color is `#867737`. This supporting tone is suitable for less prominent UI elements, chips, and secondary actions, providing visual hierarchy without competing with the primary color.

### Tertiary Color
The tertiary accent color is `#778486`. This shade is reserved for highlights, badges, and decorative elements, offering an additional layer of visual differentiation.

### Neutral Color
The neutral base color is `#7c7768`. This is used for backgrounds, surfaces, and non-chromatic elements, ensuring readability and a cohesive visual foundation.

## Typography

The design system utilizes a consistent font strategy across different text elements.

### Headlines
All headlines use the **Inter** font family, chosen for its modern aesthetic and strong readability at larger sizes.

### Body Text
Body text also uses the **Inter** font family, ensuring a harmonious typographic scale and excellent legibility for extended reading.

### Labels
Labels and smaller text elements consistently use the **Inter** font family, maintaining brand consistency even in micro-interactions.

## Shape and Form

### Roundedness
UI elements feature a **moderate** level of roundedness (value: 2). This provides a balanced aesthetic, offering a friendly touch without sacrificing professionalism or clarity.

## Spacing

### Spacing Scale
The design system employs a **normal** level of whitespace (value: 2). This ensures a good balance between information density and visual breathing room, contributing to a clean and organized layout.
